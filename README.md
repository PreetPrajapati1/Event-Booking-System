Event Booking System
Java, Spring, Hibernate, MySQL, HTML,CSS
Built a dynamic Event Booking System that allows users to browse events and book tickets
online. Implemented user registration, login authentication, and event booking functionality
using Spring Framework and Hibernate ORM for efficient database interaction. Designed a
responsive user interface with HTML(Themleaf), CSS, and Bootstrap to ensure smooth navigation and an
intuitive booking experience. Enabled CRUD operations for event management, ticket
reservations, and user profiles, improving event organization and ticket management.
